# Installation

This library provides a high-level API for Robocopy. 

## Standard Installation

You can install the core dependencies via pip:

```bash
pip install tqdm loguru
```

## Repository Setup

Ensure the `robocopy` directory from this repository is in your `PYTHONPATH`.

```bash
# Example if using in a local project
export PYTHONPATH="${PYTHONPATH}:/path/to/pyRobocopy"
```
